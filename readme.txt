== Description ==
Recent Posts List is a simple WordPress plugin to showcase the recent posts in list after each single post view.

### Key Features
* Simple, light-weight, and super-fast
* Easy to Configure
* Show Posts in List View
* Custom Title for Related/Recent posts
* Show the total posts as per your need.

== Installation ==
This section describes how to install the plugin and get it working.

e.g.

1. Upload the plugin files to the `/wp-content/plugins/recent-posts-list` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
